# AI-Smart-Detection-System
A Machine Learning based system to detect scam messages,fake resumes, and paid /free courses using Python and Scikit Learning.
Project Demonstration:
This video demonstrates the working of the AI Smart Detection System, showcasing how the model analyzes textual data and classifies content as genuine or suspicious using machine learning techniques.

## ✨ Features
- Detects scam messages
- Identifies fake resumes
- Flags misleading paid/free course advertisements
- Machine Learning based text classification

- ## 🛠 Technologies Used
- Python
- Machine Learning
- Scikit-learn
- Natural Language Processing (NLP)

## 🚀 Future Improvements
- Improve model accuracy with larger datasets
- Deploy as a web application
- Real-time scam detection system
